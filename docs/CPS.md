# Tourlinks Calculator CPS

## Context
- 골프 회원권 상품은 단순 판매가만으로 손익성을 판단하기 어렵다.
- 실제 사업성은 지역별 제휴 구장 구성, 라운드 한도, 회원/동반 가격, 이용률, 해지율, 환불금에 따라 크게 달라진다.
- 따라서 이 제품은 단순 계산기가 아니라 회원권 구조를 여러 가정으로 빠르게 검토하는 시뮬레이터여야 한다.

## Problem
- 엑셀 기반 검토는 지역별 조건과 연도별 가정을 함께 다루기 어렵고 시나리오 비교가 느리다.
- 회사 관점 지표와 고객 관점 지표를 동시에 보기 어렵다.
- 상품안이 여러 개인 경우 어떤 조합이 더 유리한지 빠르게 비교하기 어렵다.

## Solution
- 웹 기반 시뮬레이터로 회원권 구조를 시나리오 단위로 입력하고 즉시 비교한다.
- 사용자는 공통 조건과 지역별 조건을 나눠 입력한다.
- 공통 조건: 분양가, 연회비, 계약기간, 연차별 해지율, 연차별 일괄 해지 환불금
- 지역 조건: 연간 라운드 한도, 이용률, 구장 매입가, 회원가, 동반가, 비회원가, 지역 고정비, 회원 수, 동반 수
- 시스템은 다음을 계산한다.
- 회사 관점: LTV, 라운드당 마진, 회사 BEP
- 고객 관점: 라운드당 절감액, 고객 순이익, 고객 BEP
- 유지율 관점: 연도별 잔존율, 환불비용, 누적 LTV
- 여러 시나리오를 나란히 비교하고 JSON으로 저장/복원할 수 있다.

## MVP
- 공통/지역 입력 구조
- 연도별 이용률, 해지율, 환불금 입력
- 회사 BEP, 고객 BEP, LTV 계산
- 시나리오 비교
- JSON Import/Export

## Out of Scope
- 실제 계약/결제/CRM 연동
- 회원 관리 기능
- 회계 정산 및 세무 처리

## Success Criteria
- 기획 담당자가 5분 이내에 회원권 시나리오 1개를 입력하고 결과를 확인할 수 있다.
- 서로 다른 회원권안 2개 이상을 즉시 비교할 수 있다.
- 회사 손익성과 고객 가치를 동시에 보고 1차 의사결정을 내릴 수 있다.
